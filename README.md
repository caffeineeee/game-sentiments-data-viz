## Getting Started

Clone this repo:
```bash
git clone https://github.com/caxxels/game-sentiments-data-viz
```

Install the dependencies:
```bash
pnpm i
```

Run the development server:
```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
